Help

Home

The home page is used to navigate between the house alarms (left) and cameras (right).

Cameras

The cameras page provides access to all video surveillance functions.

  • Video Player - View live video footage of your home.
  • Talk button - Allows you to communicate through the surveillance cameras by clicking and holding.
  • History - Access surveillance camera history.
  • Video selector - Switch between cameras.

Alarms

The alarms page is used to access your home's alarm system.

  • Arm(stay) - Sets the alarm to stay mode.
  • Arm(away) - Sets the alarm to away mode.
  • Disarm - Used to disarm the system after it has been set.
  • View activity - View all alarm history.
  • View status - View the status of all house hold alarms.
  • Panic button - Immediately notifies local authorities.

History

The history page is used to view previously recorded surveillance footage.

  • Search bar - Search for a specific date.
  • View button - View recording.

View Activity

The view activity page is used to view previous alarm history. It shows the date, time, and the activity performed.

  • Search bar - Search for a specific date.

View Status

The view status page is used to view the status of all household fire alarms and CO2 detectors.

  • Power button - used to disable all fire alarms and CO2 detectors.
